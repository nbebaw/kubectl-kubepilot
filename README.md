# kubectl-kubepilot
Kubectl plugin called kubepilot
Usage:
  kubectl kubepilot [options]

Options:
  --cm                  : Fetch value from ConfigMap
     -c                 : ConfigMap name
     -n                 : Namespace
     -k                 : Key in ConfigMap
  --image-check         : Scan images for vulnerabilities
   -l                   : Vulnerabilitie Level
  --version             : Show version information
